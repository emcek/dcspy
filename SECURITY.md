# Security Policy

## Supported Versions

Versions of DCSpy are currently being supported with security updates.

| Version | Supported          |
|---------|--------------------|
| 3.4.x   | :white_check_mark: |
| 3.3.x   | :white_check_mark: |
| <3.2.x  | :x:                |
